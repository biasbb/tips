### sort的比较函数两种写法

> 区别在于使用自定义cmp的时候调用时不用加括号，使用仿函数调用时要加括号

- 自定义cmp

  ```c++
  bool cmp(int a, int b)
  {
      return a>b;
  }
  
  vector <int> a;
  sort(a.begin(),a.end(),cmp);
  ```

- 仿函数

  ```c++
  struct cmp
  {
      bool operator()(int a,int b)
      {
          return a>b;
      }
  };
  
  vector<int> a;
  sort(a.begin(),a.end(),cmp());
  ```

### 优先队列只能使用第二种仿函数，而且调用时不能加括号
