冒泡排序主要就是两层循环

外循环一共进行len-1次，每次得到一个最大的数放在下面，10个数的话就遍历9次。

内循环是每次遍历比较的次数，为len-i次，i是第i次遍历。10个数的话第一次遍历就比较10-1=9次。

```c++
#include <iostream>
using namespace std;
int main()
{
    int a[10];
    for(int i=0;i<10;i++)
    {
        int b;
        cin>>b;
        a[i]=b;
    }
    for(int i=0;i<9;i++)
    {
        int temp;
        for(int j=0;j<9-i;j++)
        {
           if(a[j]>a[j+1])
           {
               temp=a[j+1];
               a[j+1]=a[j];
               a[j]=temp;
           }
        }
    }
    for(int i=0;i<10;i++)
    {
        cout<<a[i]<<endl;
    }
    return 0;
}
```
